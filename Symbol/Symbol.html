Symbol

Symbols are guaranteed to be unique.
Every Symbol() call is guaranteed to return a unique Symbol.
age1 = Symbol("21") , age2 = Symbol("25") ----> age1===age2 //'false' OR Symbol('foo') === Symbol('foo') //false

You can't creat an explicit Symbol wrapper object using below syntax

const sym = new Symbol(); // TypeError

If you really want to create a Symbol wrapper object, you can use the Object() function:
const sym = Symbol('foo'); typeof sym // "symbol" const symObj = Object(sym); typeof symObj // "object"
